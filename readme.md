# Typescript

런타임 오류방지 및 안정성 제공.

typescript는 최종적으로 javascript로 변환됨. 노드, 브라우저는 기본적으로 javascript로 실행 가능. 컴파일러(tsc)로 javascript로 변환 필요.


## Setup

### npm init -y
package.json 설치

### npm i typescript
typescript 설치

### npx tsc --init
tsconfig.json 설치

### npx tsc --noEmit
타입 검사

### npx tsc
ts -> js 변환, js 파일 생성
